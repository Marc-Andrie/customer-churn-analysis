# Customer Churn Analysis

## Overview
Analyzed 7,043 telecom customer records to identify churn drivers 
and predict at-risk customers.

## Tools Used
- Python (Pandas, Scikit-learn, Matplotlib)
- SQL (SQLite)
- Power BI

## Key Findings
- Month-to-month customers churn at 43% vs 3% for two-year contracts
- Churned customers pay 21% more per month on average ($74 vs $61)
- Prediction model achieved 82% accuracy
